# Real Estate Management System

A full-stack real estate management system with Spring Boot backend, Symfony web frontend, and Flutter mobile app.

## Architecture

- **Backend**: Java Spring Boot REST API
- **Frontend Web**: Symfony (PHP)
- **Frontend Mobile**: Flutter (Dart)
- **Database**: SQL Server Express (local) / SQL Server (production)

## Project Structure

```
real-estate-system/
├── backend/          # Spring Boot REST API
├── frontend-web/     # Symfony Web Application
├── frontend-mobile/  # Flutter Mobile App
└── docs/             # Documentation
```
