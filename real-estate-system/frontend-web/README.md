# Real Estate Frontend Web

Symfony-based web frontend for the Real Estate Management System.

## Prerequisites

- PHP 8.2+
- Composer
- Symfony CLI (optional)

## Setup

1. Run `composer install`
2. Configure `.env` with API URL
3. Start with `symfony server:start` or `php -S localhost:8000 -t public`

## Features

- Property listing and management
- User authentication via backend API
- Responsive web interface
