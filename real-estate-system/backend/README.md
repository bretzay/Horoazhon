# Real Estate Backend API

Spring Boot REST API for the Real Estate Management System.

## Prerequisites

- Java 17+
- Maven 3.8+
- SQL Server Express (local)

## Setup

1. Configure database connection in `src/main/resources/application.properties`
2. Run `mvn clean install`
3. Start with `mvn spring-boot:run`

See the API documentation in `/docs/API_DOCUMENTATION.md`
